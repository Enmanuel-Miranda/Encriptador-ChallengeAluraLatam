# Challenge 1 - Desarrollo Frontend

Este proyecto es el resultado del primer desafío técnico de desarrollo web, enfocado en la creación de una interfaz limpia, interactiva y completamente adaptable a dispositivos móviles.

**🔗 Enlace al proyecto:** [Visitar sitio en producción](https://enmanuel-miranda.github.io/challenge1enmanuel.github.io)

---

## ⚙️ Funcionalidades clave

- **Diseño Adaptativo (Responsive):** La interfaz se ajusta correctamente a pantallas de celulares, tablets y computadoras.
- **Interactividad dinámica:** Uso de lógica en el cliente para mejorar la experiencia del usuario (UX).
- **Estructura semántica:** Código organizado bajo las mejores prácticas de maquetación.

## 🛠️ Tecnologías utilizadas

- **HTML5** / **CSS3**
- **JavaScript (ES6+)**
- **GitHub Pages** (Alojamiento y despliegue)
